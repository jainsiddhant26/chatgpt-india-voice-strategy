# Customer Feedback Survey Data

This folder contains the raw Google Form survey responses (`Customer-Feedback.xlsx`) collected from **60 Indian college students (18–24)** in January 2026.

## Survey Overview
- **n = 60** respondents
- **Segment:** Indian college students, frequent ChatGPT mobile users
- **Focus:** Voice input adoption, blockers, and willingness to try

## Key Findings from Raw Data

| Question | Top Response |
|----------|--------------|
| Why don't you use voice? | "I prefer typing because I can see and fix my mistakes" |
| Would you try voice if you could see & edit transcription? | 78.4% said Yes / Maybe |
| Have you tried voice before? | 48% tried and paused, 25% never tried |
| What stops you? | 48% — Can't see/edit before sending |

> The raw `.xlsx` file with all 60 responses is included in this folder.
